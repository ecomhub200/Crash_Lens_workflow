# CrashLens Wiki

This is a mirror of the Obsidian wiki for the CrashLens project.
Source of truth: Obsidian vault on local machine.

## Structure
- `concepts/` — Architecture, pipeline, schema documentation
- `entities/` — States, tools, infrastructure
- `analyses/` — Cross-cutting analysis pages
- `sources/` — Source repo documentation

## Sync
- Obsidian MCP updates → git commit + push from GitHub Desktop
- Claude Code updates → git pull into Obsidian vault
- Keep both in sync after any changes
