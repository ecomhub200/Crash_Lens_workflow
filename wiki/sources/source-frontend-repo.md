## What This Is

JavaScript/Node.js web application repository for the Crash Lens platform. Browser-based SPA with 12+ analysis tabs, Node.js backend proxy, Docker deployment.

## Key Takeaways Extracted

- Monolithic `index.html` (151,205 lines) being migrated to modular architecture (`app/modules/`)
- 12+ analysis tabs covering the full safety analysis workflow
- Claude AI assistant integrated for natural language crash analysis
- Commercial-ready: Firebase auth, Stripe billing, Coolify deployment
- Rich visualization: Leaflet maps, Chart.js, PMTiles, Turf.js
- Report generation: jsPDF (PDF) and DOCX.js (Word)
